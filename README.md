# AEH Portfolio (SPA with pages via Vue Router)

This is a **single-page app** (SPA). You *won’t* see multiple HTML files.
The “pages” are Vue components in `src/views/` and are routed via Vue Router:

- `/` Home
- `/projects` Projects
- `/about` About
- `/contact` Contact

## Run
```bash
npm install
npm run dev
```

## Edit
- Projects: `src/data/projects.json`
- Socials: `src/config/socials.js`
